# Test Brain

This is a software implementation of the Zauberzeug Test Brain.

# Installation

- Add the config for your Robot Brain, for the pins refer to pindoc.

- Sync the repo using `livesync <robot>`

- Create a env `echo "BRAIN_ID={robot_brain_id}" > .env`

- Create the `.rosys` folder with `mkdir ~/.rosys`

- Start the docker by running `./docker.sh U` inside your testbrain folder.
